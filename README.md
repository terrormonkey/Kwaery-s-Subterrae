# (Eng) 
Installation: Navigate to Releases and download the provided .zip file. Extract the contents to a secure location on your system. Note: Do not move or delete this folder after installation, as the extension relies on this path; otherwise, a reinstallation will be required.

To manually add the extension to Chrome:

Open Chrome and enter chrome://extensions/ in the address bar.

Enable "Developer mode" using the toggle in the top-right corner.

Click "Load unpacked" and select the folder containing your extracted files.

# (Ger) 
Installation: Gehe zu Releases, lade die .zip-Datei herunter und entpacke den Inhalt an einem sicheren Ort auf deiner Festplatte. Hinweis: Der Ordner darf nach der Installation weder verschoben noch gelöscht werden, da Chrome den Pfad statisch speichert; andernfalls muss die Erweiterung neu installiert werden.

So fügst du die Extension manuell in Chrome hinzu:

Öffne Chrome und gib chrome://extensions/ in die Adresszeile ein.

Aktiviere oben rechts den „Entwicklermodus“ (Developer mode).

Klicke auf „Entpackte Erweiterung laden“ (Load unpacked) und wähle den Ordner aus, in dem die Dateien liegen.
